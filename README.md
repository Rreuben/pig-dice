# Pig, The Dice Game
#### A game which in its basic form is playable with just a single die.

## Description
Pig is considered a jeopardy game, where you risk everything to see if you can win more. The numbers on the die (except for "1") each represent points that can accumulateâ€”the winner is the first player to reach 50 points or more.
